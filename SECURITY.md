# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in this project, please **do not** open a public issue.

Instead, contact me directly:
- Telegram: https://t.me/anime_24hr

I will respond as soon as possible and fix the issue promptly.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅ |

## Security Best Practices

- Never share your bot token publicly
- Never commit `.env` files to GitHub
- Regenerate credentials immediately if exposed
